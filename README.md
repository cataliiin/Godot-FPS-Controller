# Godot FPS Controller

### Features

- all basic movement features like *running, jumping and crouching*
- solves slope sliding issues

### Controls

**W** - move forward \
**A** - move left \
**S** - move back \
**D** - move right \

**SPACE** - jump \
**L_SHIFT** - run \
**C** - crouch \
**`** - toggle mouse mode ( *the button under esc* )
